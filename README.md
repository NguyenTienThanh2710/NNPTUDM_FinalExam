# 📋 Các công việc cần thực hiện

## 1. Backend (NodeJS - RESTful API)
- Thiết kế cơ sở dữ liệu cho hệ thống cửa hàng điện thoại  
- Xây dựng RESTful API bằng NodeJS  
- Tổ chức project theo cấu trúc folder đã học  

### Chức năng API
- API đăng ký / đăng nhập  
- API quản lý sản phẩm (thêm, sửa, xoá, xem danh sách)  
- API quản lý khách hàng (thêm, sửa, xoá, xem danh sách)  
- API quản lý đơn hàng (tạo đơn hàng, cập nhật, xem danh sách)

---

## 2. Frontend
- Xây dựng giao diện hệ thống  
- Kết nối và lấy dữ liệu từ RESTful API  
- Hiển thị danh sách sản phẩm  
- Thêm / sửa / xoá sản phẩm  
- Hiển thị thông tin khách hàng  
- Hiển thị danh sách đơn hàng
